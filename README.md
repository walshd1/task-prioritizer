# Task Prioritizer

Prioritizes tasks in a project board based on urgency, impact, and dependencies identified in issues.

## Free
```yaml
- uses: walshd1/task-prioritizer@v1
  with:
    gemini_api_key: ${{ secrets.GEMINI_API_KEY }}
```

## Paid (cost + 4.75%)
```yaml
- uses: walshd1/task-prioritizer@v1
  with:
    service_token: ${{ secrets.ACTION_FACTORY_TOKEN }}
```
